Stadia+

Stadia+ is an addon for Google's Stadia gaming platform. It includes a lot of useful features and additions that help with everything from solving network issues to managing your game library. Stadia+ is developed independently, licensed under GNU GPL v3 and has no connection to Google, Alphabet or any other Google product or subsidiary.

If you're new to Stadia+, make sure to read through our Getting Started guide.

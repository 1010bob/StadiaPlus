Network Monitor


Q: How do I open the network monitor?

A: Press Ctrl+M at any time in the Stadia menu to open the network monitor, you can also use the 'Toggle Network Monitor' button in the network section of the Stadia menu.


Q: My network monitor is stuck on 'Start a game to monitor traffic'

A: Reload the page to reset the monitor script, then open the network monitor like normal.


Q: The network monitor is stuck and refuses to update

A: Sometimes, the script monitoring traffic stops working, reload the page to reset it.

Force Codec


Q: How do I force my codec?

A: Use the dropdown found in the network section of the Stadia menu.


Q: My network section is empty

A: It takes a short while for the network section to load, please wait up to 30

seconds.
